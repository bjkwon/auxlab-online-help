line

y = line ( x,y[=""] )
y = line ( handle,x,y[=""] )
y = handle.line ( x,y[=""] )

Generate a line object in the designated axes

arg

type

Description

Unit or Value scale

handle

HGO

handle to the axes

x

VECT or AUD

x-data

y

VECT or AUD

y-data

Outputs
  • y is the handle to the line object

Notes
  • This is a low-level version of the plot function.
  • This simply adds a line object to the axes, as opposed to following the "nextplot" property in the plot function.
  • If handle is omitted, this function creates a new figure window and plots
  • If y is specified, y is plotted as a function of x.
  • If y is omitted, x is plotted with the index (if non-audio) or the time (if audio) on the x-axis.
