z = x <> y

Change the duration of an audio signal x by a factor of y without changing the pitch

Commutative

No

Data Types

x

y

z

AUD

SCAL

AUD

Notes
  • This is equivalent to z=timestretch(x,y) or z=x.timestretch(y)
  • See the function description for timestretch for more information.

Examples
  • A tone glide from 400 Hz to 1000 Hz with arbitrary amplitude fluctuations.

AUX> x=tone([400 1000],1000).sqrt@-3
AUX> tseq=[0 .2 .3 .7 .8 1;][-100 -6 -30 -10 -5 0];
AUX> y=(x@tseq).ramp(20)

AUX> z=y<>.8
AUX> figure("z").children.x.lim=[0 1] // to adjust the xlim the same as y
